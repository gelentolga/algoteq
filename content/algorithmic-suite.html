Algorithm Suite

“ The best way to predict the future is to create it. ” — Peter Drucker

Data and Analysis

Our suite leverages a managed historical data repository per stock, enabling tailored default behaviour and effective parameterisation. Market data is enhanced in real time using advanced analytics to improve algorithm responsiveness and decision-making.

Custom aggregation, venue simulation, and synthetic data enhancements are standard — ensuring smart routing and execution edge.

Available Algorithms

The suite includes a variety of algorithms, each categorized for specific trading needs:

  • Profile — VWAP, TWAP, IS, POV, MC
  • Passive — PEG, ICEBERG
  • Responsive — SNIPER, TOUCH, STOP, LIMIT-TO-MARKET
  • Liquidity Seeking — Custom, hidden-liquidity focused, and alpha-optimised algorithms with smart routing

Simulation

Developers and quants can simulate algorithms on historical data with no code change. Visualize performance, tweak parameters, and analyse execution in both live and post-trade contexts.

Simulation outputs also train our parameter response models, supporting rapid iteration and optimisation of trading strategies.

Summary

The Algoteq Algorithm Suite is a powerful, modular framework that supports trading innovation and adaptability. With seamless integration of strategy, data, and simulation, it empowers traders to evolve with the market — across any asset class.
